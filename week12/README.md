# Задачи

В задача 1 и 2 трябва да правите проверка на входа

## Задача 1


Напишете следните функции

- `int** genMatrix(size_t n, size_t m)` - генерира матрица с числа с размер NxM

- `int** genTriangle(size_t n)` :- генерира триъгълник с числа с размер на страните NxNxN√2


## Задача 2

Напишете функции за изтриване на двумерните масиви

- `void freeMatrix(int***, size_t n, size_t m)`

- `void freeTriangle(int***, size_t n, size_t m)`

## Задача 3


Напишете следните функции

- `int** mulMatrix(int**, int a, size_t n, size_t m)` - умножава матрицата по скалар

- `void printMatrix(int**, size_t n, size_t m)`

## Задача 4

Напишете следните функции

- `void printTriangle(int**, size_t n)` - принтира триъгълника

- `long long int squashTriangle(int**, size_t n)` - сумира всички елементи на триъгълника

## Задача 5

Напишете следните функции

- `int** sumMatrix(int**, int**, size_t n, size_t m)` - събира две матрици

- `int** mulMatrix(int**, int**, size_t n, size_t m, size_t k)` - умножава две матрици

## Задача 6

С помощта на функциите които досега сте писали и рекурсич напишете програма в която потребителя въвежда матрица от 0, 1, 2 и 3 където 

0. означава празно място

1. означава стена

2. е плод, с който може вашето човече да прави два хода

3. е гамен, който кара вашето човече да не може да се движи

За един ход вашето човече може да прави стъпка нагоре, надолу, наляво и надясно. Напишете следните 2 функции

- `bool findPath(int** labyrinth, size_t n, size_t m, int* path)` - намира дали има път и го пази в масив с терминираща нула

- `bool findOptimumPath(int** labyrinth, size_t n, size_t m, int* path)` - намира дали има оптимален път и го пази в масив с терминираща нула
