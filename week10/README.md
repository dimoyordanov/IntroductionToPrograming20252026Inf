# Задачи
## Задача 1
Имеплементирайте следните функци
- `void fill(int arr[100], int а, int d)`, която пълни всеки елемент на масива по следната процедура. На клетка с иденкс i стойноста е $a+d\times i$
- `void zero(int arr[100])`, която нулира всеки елемент на масива
- `void ones(int arr[100])`, която поставя единица на всеки елемент на масива
## Задача 2
Напишете функциите
- `int linearSearch(int arr[100], int key)` - търси дали има ключа в масива, ако ли не връща разликата му с най-близкия елемент
- `int isSorted(int arr[100])` - гледа дали масива е сортиран в монотонно растаящ ред, ако не е сортиран връща първата срещната разлика на двата елемента, които не са сортирани.
## Задача 3
Напишете функциите, като имплементирате максимално много проверки.
- `long long int getElem(int values[100], int index)` - взима елемент на даден индекс
- `long long int setElem(int values[100], int index, int value)` - поставя value на елемента на даден индекс
## Задача 4
Имплементирайте функциите за непълен масив.
- `int addFront(int arr[100], int size, int value)` - върнете 0 ако е станало успешно, 1 ако няма място
- `int addBack(int arr[100], int size, int value)` - върнете 0 ако е станало успешно, 1 ако няма място
## Задача 5
Имплементирайте следните функции
- `void removeFront(int arr[100], int size, int value)` - Върнете 1 ако махането не е успешно, 0 ако е успешно.
- `void removeBack(int arr[100], int size, int value)` - Върнете 1 ако махането не е успешно, 0 ако е успешно.
## Задача 6
Напишете следните функции
- `void insertionSort(int arr[SIZE], int size)` - сортирайте чрез вмъкване.
- `void bubbleSort(int arr[SIZE], int size)` - сортирайте чрез метода на мехурчето

- `binarySearch(int arr[100], int value)`
